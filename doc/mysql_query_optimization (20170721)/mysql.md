# MySQL体系结构 #

![MySQL](./MySQL.png)

1. Connectors指的是不同语言中与SQL的交互

4. SQL Interface: SQL接口

5. Parser: 解析器

6. Optimizer: 查询优化器

7. Cache和Buffer: 查询缓存

8. Engine: 存储引擎
